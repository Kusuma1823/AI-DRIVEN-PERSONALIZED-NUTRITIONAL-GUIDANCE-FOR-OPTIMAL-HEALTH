# 🥗 Personalized Nutritional Guidance for Optimized Health

An AI-powered web application that analyzes food product ingredients and provides personalized nutritional recommendations based on a user's health profile. The system combines **Machine Learning**, **OCR**, and the **MERN Stack** to help users make healthier food choices.

---

## 📌 Features

- 🔐 User Authentication (Login & Registration)
- 📷 OCR-based Food Label Text Extraction
- 🥫 Ingredient Analysis
- 🤖 Machine Learning-based Health Prediction
- ⚠️ Harmful Ingredient Detection
- 🩺 Personalized Nutritional Recommendations
- 🔍 Product Search & Analysis
- 📊 Interactive User Dashboard

## 🚀 Project Workflow

1. 👤 User logs into the application.
2. 📤 Uploads or searches for a food product.
3. 📷 OCR extracts nutritional information from the food label.
4. 🤖 Machine Learning analyzes the ingredients.
5. ⚠️ Harmful ingredients are identified.
6. 🩺 Personalized dietary recommendations are generated.
7. 📊 Results are displayed through an interactive dashboard.
