# Personalized Nutritional Guidance for Optimized Health

## Overview
Personalized Nutritional Guidance for Optimized Health is an AI-powered web application that analyzes food product ingredients and provides personalized dietary recommendations based on a user's health profile. The system uses Optical Character Recognition (OCR) and Machine Learning to extract nutritional information from food labels, identify potentially harmful ingredients, and generate health-focused insights.

## Features
- User Authentication (Login & Registration)
- OCR-based Food Label Text Extraction
- Ingredient Analysis
- Personalized Health Recommendations
- Harmful Ingredient Detection
- Machine Learning-based Prediction
- User-Friendly Dashboard
- Search and Filter Products

## Project Workflow
1. User logs into the application.
2. Uploads or searches for a food product.
3. OCR extracts nutritional information from the food label.
4. The system analyzes ingredients using Machine Learning.
5. Harmful ingredients are identified.
6. Personalized dietary recommendations are generated based on the user's health conditions.
7. Results are displayed through an interactive dashboard.

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Personalized-Nutritional-Guidance.git
```

Navigate to the project folder:

```bash
cd Personalized-Nutritional-Guidance
```

Install frontend dependencies:

```bash
npm install
```

Install backend dependencies:

```bash
npm install
```

Start the application:

```bash
npm start
```
